TLE 轨道根数查询

GET /api/v1/tle/{norad_cat_id}

适用场景 根据 NORAD 编号获取指定空间目标的最新 TLE（两行轨道根数）。返回标准 TLE 文本和解析后的六个经典轨道根数，可直接用于 SGP4 传播或外部轨道分析工具。

请求参数

参数名位置类型必填说明
norad_cat_id路径参数int是NORAD 目标编号（5 位数字）

响应字段

字段类型说明
norad_cat_idintNORAD 编号
namestring目标名称
object_typestring目标类型
countrystring国家代码
tle_line1stringTLE 第一行原始文本（69 字符）
tle_line2stringTLE 第二行原始文本（69 字符）
epoch_utcstringTLE 历元时刻 (ISO-8601)
elements.inclination_degfloat轨道倾角 (°)
elements.eccentricityfloat偏心率
elements.mean_motion_rev_dayfloat平均运动 (rev/day)
elements.raan_degfloat升交点赤经 (°)
elements.arg_pericenter_degfloat近地点幅角 (°)
elements.mean_anomaly_degfloat平近点角 (°)
elements.bstarfloatB* 阻力系数
orbit.perigee_kmfloat近地点高度 (km)
orbit.apogee_kmfloat远地点高度 (km)

curl 示例

# 查询国际空间站 (ISS) 的 TLE
curl http://<your-host>:8000/api/v1/tle/25544

💡 TLE 时效性：返回的是数据库中最新的 TLE。TLE 的年龄（当前时间 - epoch）越大，SGP4 传播误差越大。通常 TLE 年龄应小于 7 天以获得可接受的位置精度。

错误响应

当指定的 NORAD ID 不存在于数据库中时：

{
  "error": "未找到 NORAD 99999 的轨道根数数据",
  "norad_cat_id": 99999
}
