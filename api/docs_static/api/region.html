区域碎片查询

POST /api/v1/debris/region

适用场景 查找指定地理坐标周围一定范围内、特定高度区间内的在轨空间目标。适用于发射前空域安全评估、区域态势监控。

请求参数

参数名类型必填默认值说明
lat_degfloat是—搜索中心纬度 (°)，范围 -90 到 90
lon_degfloat是—搜索中心经度 (°)，范围 -180 到 180
radius_kmfloat否500搜索半径 (km)
alt_min_kmfloat否0最低轨道高度 (km)
alt_max_kmfloat否2000最高轨道高度 (km)
object_typestring否"ALL"目标类型：ALL / DEBRIS / PAYLOAD / ROCKET BODY
t_start_utcstring否当前时刻时间窗口起点，ISO-8601 格式
hoursfloat否6时间窗口长度（小时）
limitint否50最大返回条数，1–200

响应字段

字段类型说明
regionobject搜索参数回显（中心坐标、半径、高度范围、时间窗口）
countint匹配的总目标数量
showingint本次实际返回的数量
objectsarray匹配的目标列表
objects[].norad_cat_idintNORAD 编号
objects[].namestring目标名称
objects[].object_typestring类型：DEBRIS / PAYLOAD / ROCKET BODY
objects[].countrystring国家代码
objects[].perigee_kmfloat近地点高度 (km)
objects[].apogee_kmfloat远地点高度 (km)
objects[].inclination_degfloat轨道倾角 (°)
objects[].eccentricityfloat偏心率
objects[].approx_dist_kmfloat与搜索中心的近似距离 (km)
summarystring人类可读的中文摘要
methodstring查询方法：PostGIS 或 SGP4

curl 示例

curl -X POST http://<your-host>:8000/api/v1/debris/region \
  -H "Content-Type: application/json" \
  -d '{
    "lat_deg": 19.61,
    "lon_deg": 110.95,
    "radius_km": 500,
    "alt_min_km": 200,
    "alt_max_km": 2000,
    "object_type": "DEBRIS",
    "hours": 6,
    "limit": 10
  }'
