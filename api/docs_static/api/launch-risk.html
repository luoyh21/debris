发射碰撞风险预测

POST /api/v1/launch/risk

适用场景 模拟火箭发射全过程（6-DOF 轨迹仿真），并对各飞行阶段进行碎片碰撞概率评估。集成了轨迹仿真、碎片检索、TCA 求解、Foster Pc 计算的全链路流程。

⚠️ 注意：此接口计算量较大，响应时间通常需要 10–60 秒。适合后台调用或允许较长等待的场景。

请求参数

参数名类型必填默认值说明
vehiclestring否"CZ-5B"运载火箭型号：CZ-5B / Falcon9 / Ariane6
launch_lat_degfloat否19.61发射场纬度 (°)
launch_lon_degfloat否110.95发射场经度 (°)
launch_az_degfloat否90发射方位角 (°)，0=正北，90=正东
launch_utcstring否明日 06:00发射时间 ISO-8601 UTC
t_max_sfloat否3600仿真最长时间 (秒)，600–7200
include_demo_threatsbool否true是否包含演示用合成碎片目标

响应字段

字段类型说明
assumed_paramsobject系统自动推断或默认的参数值
vehicle_usedstring实际使用的火箭预设名称
trajectoryobject关键时刻的高度/速度/质量摘要
phasesarray各飞行阶段的风险评估结果
phases[].namestring阶段名称（ASCENT/PARKING_ORBIT/TRANSFER_BURN/POST_SEPARATION）
phases[].duration_sfloat阶段持续秒数
phases[].candidatesint空间预筛选的候选碎片数
phases[].max_pcfloat该阶段最大碰撞概率
phases[].risk_levelstring风险等级：GREEN / YELLOW / AMBER / RED
top_eventsarray最高 Pc 的前 10 条合取事件
overall_riskstring整体风险等级
overall_risk_cnstring整体风险等级（中文）
max_pcfloat全轨迹最大 Pc 值
recommendationstring中文规避建议
caveatsarray假设和限制说明列表

curl 示例

curl -X POST http://<your-host>:8000/api/v1/launch/risk \
  -H "Content-Type: application/json" \
  -d '{
    "vehicle": "CZ-5B",
    "launch_lat_deg": 19.61,
    "launch_lon_deg": 110.95,
    "launch_az_deg": 90,
    "launch_utc": "2026-04-20T06:00:00",
    "t_max_s": 3600,
    "include_demo_threats": true
  }'
