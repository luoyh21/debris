再入预报

POST /api/v1/debris/reentry

适用场景 预测即将再入大气层的空间目标。基于 Space-Track 提供的 decay_date（确认再入日期）和近地点高度进行筛选，帮助跟踪可能产生地面风险的大型碎片和箭体再入事件。

请求参数

参数名类型必填默认值说明
days_aheadfloat否30预报窗口天数，1–365
alt_max_kmfloat否300近地点高度阈值 (km)——低于此值且无确认 decay_date 的目标也会被纳入
object_typestring否"ALL"目标类型过滤：ALL / DEBRIS / PAYLOAD / ROCKET BODY
limitint否50最大返回条数，1–200

响应字段

字段类型说明
windowobject查询参数回显（预报天数、高度阈值、时间范围）
countint匹配总数量
showingint本次返回数量
objectsarray目标列表
objects[].norad_cat_idintNORAD 编号
objects[].namestring目标名称
objects[].object_typestring目标类型
objects[].countrystring国家代码
objects[].launch_datestring发射日期 (ISO-8601)
objects[].decay_datestring|null确认的再入日期（null 表示尚无确认日期）
objects[].days_to_reentryfloat|null距离再入的天数
objects[].perigee_kmfloat近地点高度 (km)
objects[].apogee_kmfloat远地点高度 (km)
objects[].inclination_degfloat轨道倾角 (°)
summarystring中文摘要

curl 示例

curl -X POST http://<your-host>:8000/api/v1/debris/reentry \
  -H "Content-Type: application/json" \
  -d '{
    "days_ahead": 30,
    "alt_max_km": 300,
    "object_type": "ROCKET BODY",
    "limit": 20
  }'
