空间碎片监测与发射碰撞风险评估系统

本系统是一套面向航天发射任务规划人员和空间碎片研究人员的全链路空间态势感知平台。从数据采集、轨道预报，到碰撞风险评估和发射窗口优化，提供端到端的决策支持。

系统整合了 Space-Track、UCS、ESA DISCOS、GCAT、UNOOSA 五大权威数据源，统一去重后形成 60,000+ 在轨目标的综合目录，并提供 8 个交互式功能页面和 6 个 REST API 接口。

核心功能一览

系统概览面板

一目了然掌握全库在轨目标总量、各类型分布、数据摄入状态和发射历史趋势。

可视化探索

正射球面和 3D 地球上实时观看碎片运动，高度分层下钻，区域碎片搜索，轨道预报。

目标目录

多源融合的统一目标数据库，支持按数据源、国家、类型、RCS 等维度筛选与检索。

轨迹仿真

CZ-5B / Falcon 9 / Ariane 6 火箭 6-DOF 轨迹仿真，3D 可视化飞行全过程。

LCOLA 飞越筛选

扫描 ±N 分钟发射窗口，找出禁发时段，推荐最安全发射时刻。

碰撞风险评估

对发射全过程逐阶段计算碰撞概率 (Pc)，四级风险预警 + 规避建议。

长期风险评估

基于泊松统计和 ORDEM/MASTER 通量模型，预测卫星寿命期内的累积碰撞风险。

AI 智能助手

用自然语言提问，AI 自动调用分析工具返回结构化分析结果。
