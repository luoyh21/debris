轨迹仿真

场景 模拟火箭从发射到入轨的完整飞行过程，生成 6-DOF 轨迹数据，作为后续碰撞风险评估和 LCOLA 飞越筛选的输入。

仿真基于 ECEF 坐标系的 6 自由度常微分方程（J2 引力 + USSA-76 大气阻力 + 推力矢量 + 重力转弯俯仰程序），生成高精度的火箭飞行轨迹。

⚡ 重要：轨迹仿真是碰撞风险评估和 LCOLA 飞越筛选的前置步骤。请先在本页面运行仿真，生成轨迹数据后再前往对应模块。

使用步骤

  1. 在左侧导航栏点击 「轨迹仿真」
  2. 在仿真参数表单中设置火箭型号、发射场坐标、方位角、发射时间等参数
  3. 点击 「运行仿真」 按钮
  4. 等待 5–20 秒（首次运行较慢），仿真完成后页面自动显示结果
  5. 查看飞行阶段摘要、高度/速度曲线、3D 轨迹等输出
  6. 仿真数据将自动保存到会话中，可直接跳转到碰撞风险评估或 LCOLA 页面

参数设置

参数说明选项/默认值
运载火箭选择预设火箭型号CZ-5B（长征五号B）/ Falcon9（猎鹰九号）/ Ariane6（阿丽亚娜6号）
发射时间 (UTC)发射时刻，ISO-8601 格式如 2026-04-15T06:00:00
发射方位角 (°)发射方向，0°=正北，90°=正东默认 90°（正东发射）
发射场纬度 (°)发射场地理纬度默认 19.61°（文昌）
发射场经度 (°)发射场地理经度默认 110.95°（文昌）
仿真时长 (s)积分最长持续秒数默认 3600 秒（1 小时）
输出步长 (s)轨迹点的时间间隔默认 10 秒

输出结果

📋 飞行阶段摘要

数据表

系统自动将飞行轨迹划分为多个阶段，并输出每个阶段的关键参数：

阶段英文名说明
上升段ASCENT从发射到一级主发动机关机 (MECO1)
停泊轨道段PARKING_ORBIT滑行段，等待变轨窗口
变轨推进段TRANSFER_BURN二级发动机点火推入目标轨道
分离后脱轨段POST_SEPARATION载荷分离后的自由飞行

每个阶段显示：MET 起止时间、近地点/远地点高度范围、平均速度、风险特征。

📈 高度与速度曲线

图表

两幅并排折线图展示火箭从发射到入轨全过程中的高度变化和速度变化。横轴为 MET（任务经过时间，秒），纵轴分别为高度 (km) 和速度 (km/s)。

⚖️ 质量消耗曲线

图表

展示火箭总质量随时间的变化。阶梯式下降对应一级关机/级间分离事件。

📄 状态向量数据表

可展开

展开「状态向量数据表」可查看前 100 行的详细数据：MET、高度、纬度、经度、ECI 速度分量 (vx, vy, vz)、质量。
