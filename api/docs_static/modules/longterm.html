长期风险评估

场景 评估一颗卫星在其运营寿命期间（数年到数十年）的累积碰撞风险。回答「在某个轨道高度运营卫星 N 年，碰撞概率有多高？需要预留多少规避燃料？」

与碰撞风险评估（针对单次发射）不同，长期风险评估关注的是卫星在轨运营期间的统计性碰撞累积风险，基于实际碎片环境密度和泊松随机过程进行蒙特卡洛模拟。

使用步骤

  1. 在左侧导航栏点击 「长期风险评估」
  2. 设置轨道参数（高度、倾角）和任务参数（运营年限、卫星截面积等）
  3. 点击 「运行评估」 按钮
  4. 系统从数据库提取目标轨道高度附近的碎片环境数据
  5. 运行蒙特卡洛模拟，计算累积碰撞概率
  6. 查看输出指标和图表

参数调整建议

💡 如何降低长期碰撞风险

如果评估结果显示风险偏高，可以考虑以下调整：

  • 调整轨道高度：避开碎片密度最高的高度带（如 750–900 km）
  • 缩短运营年限：碰撞概率随时间近似线性增长，缩短寿命可显著降低风险
  • 减小卫星截面积：碰撞概率与截面积成正比
  • 选择低密度轨道倾角：太阳同步轨道（~97°）碎片密度较高，可考虑略微偏离
  • 增加规避燃料预算：确保有足够的 ΔV 应对预期的接近事件
💡 提示：长期风险评估也可通过 AI 助手调用。尝试输入「我计划在 800km 高度（倾角 53°）运营一颗卫星，寿命 5 年，预测碰撞风险和建议规避燃料预算」。
