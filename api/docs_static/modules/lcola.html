LCOLA 飞越筛选

场景 为发射任务找到最安全的发射时刻——在一段时间窗口内扫描所有候选发射时刻，找出哪些时刻会触发碎片碰撞风险（禁发窗口），推荐安全发射时段。

⚡ 前置条件：LCOLA 飞越筛选需要使用轨迹仿真的结果。请先在 轨迹仿真 页面运行仿真并生成轨迹数据。

LCOLA 是什么

LCOLA（Launch Collision Avoidance）是发射碰撞规避分析的简称。其核心思想是：对于一个确定的发射轨迹模板，通过微调发射时刻（前后偏移几十分钟），火箭到达各空间位置的时间会发生变化，从而改变与在轨碎片的空间关系。LCOLA 筛选通过逐时刻评估碰撞概率，找出所有会产生高碰撞风险的发射时刻（禁发窗口），帮助规划人员选择安全时刻发射。

使用流程

  1. 加载轨迹：确保已在 轨迹仿真 页面运行仿真。返回 LCOLA 页面后，系统自动加载当前轨迹。
  2. 设置筛选参数：在「筛选配置」面板中调整窗口宽度、步长、HBR 等参数。
  3. 启动筛选：点击 「开始飞越筛选（后台运行）」。筛选在后台执行，您可以切换到其他页面，完成后右上角会弹出通知。
  4. 查看结果：返回 LCOLA 页面，查看禁发窗口、安全窗口和高风险合取事件。

筛选参数

参数说明默认值
载人任务勾选后 Pc 门限从 10⁻⁵ 降低到 10⁻⁶（更严格）未勾选
联合硬体半径 HBR (m)火箭和碎片的外接球半径之和20 m
窗口宽度（分钟）以标称发射时刻为中心，向前后各扩展的时间范围±30 分钟
筛选步长（秒）相邻两个候选发射时刻之间的间隔60 秒
注入演示威胁注入合成的禁发窗口用于演示。碎片数据库较小时推荐开启开启
💡 耗时预估：页面会自动估算本次筛选需要评估多少个候选时刻及预计耗时。例如 ±30 分钟窗口、60 秒步长 = 61 个候选时刻，在碎片数据库已填充时约需 15–60 秒。步长越小精度越高但耗时越长。

结果解读

📈 Pc 时间曲线

图表

横轴为相对于标称发射时刻的偏移量（秒），纵轴为该时刻发射对应的最大碰撞概率 (Pc)。红色阴影区域标记了超过 Pc 门限的禁射时段。

🚫 禁射窗口（Blackout Windows）

数据表

列出所有被标记为禁发的时间段。每个窗口包含开始时间(UTC)、结束时间(UTC)和持续时长(分钟)。如果窗口内无禁射时段，将显示「全时段可发射」。

安全发射窗口（Safe Windows）

数据表

禁射窗口之外的连续安全时段。任务规划人员应从这些窗口中选择发射时刻。

高风险合取事件（Top 50）

数据表

按碰撞概率 (Pc) 降序排列的前 50 条合取事件。每条记录包含：

字段含义
风险等级RED / AMBER / YELLOW / GREEN
NORAD ID碎片的编号（合成威胁显示为「合成 #n」）
目标名称碎片名称
飞行阶段碰撞发生在哪个飞行阶段
TCA (UTC)最近接时刻
最近距离 (km)两目标间的最小距离
Foster Pc碰撞概率，科学计数法
相对速度 (km/s)相遇时的相对速度
偏移量 (s)相对于标称发射时刻的偏移量

📊 按飞行阶段统计

图表

柱状图展示各飞行阶段（上升段/停泊轨道段等）的合取事件数量，帮助判断哪个阶段的碰撞风险最集中。

💡 后台运行说明：LCOLA 筛选在后台线程中执行，您可以在计算过程中切换到其他页面继续工作。侧边栏会显示当前计算进度，完成后右上角弹出通知。如需中途取消，点击「取消」按钮。
