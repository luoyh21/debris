系统概览面板

场景 快速了解系统数据状态和空间环境态势，获取全库统计数据的全景视图。

概览面板是您登录系统后的默认首页。它汇总了所有数据源的关键指标，帮助您在几秒内掌握当前空间环境的整体状况。

使用步骤

  1. 在左侧导航栏点击 「系统概览」（默认已选中）
  2. 查看顶部四张指标卡片，了解当前在轨目标的总量和分类统计
  3. 向下滚动，依次浏览各个统计图表和数据矩阵
  4. 需要深入查看某类数据时，点击对应模块链接跳转

数据摄入状态矩阵

📥 多源数据摄入状态

监控

以「数据源 × 目标类型」交叉矩阵形式展示各数据源已入库的记录数量。每行代表一个数据源（Space-Track、UCS、ESA-DISCOS），每列代表一种目标类型（载荷/碎片/箭体/未知），末列为合计。

矩阵下方显示融合后的数据源总数和总目标数。如果某个数据源的计数为 0，说明该数据源尚未完成数据摄入。

展开「补充与专题数据源」可查看 GCAT、UNOOSA 的聚合统计记录数，以及 Asterank 小行星 / 近地天体专题库的入库条数。

💡 提示：概览面板的数据每 60 秒自动刷新一次。如果您刚完成数据摄入操作，稍等片刻即可看到更新的统计数字。
