可视化探索

场景 通过沉浸式 3D 地球可视化，实时观察空间碎片在轨运动、探索不同高度层的目标分布、追踪单个目标的轨道预报，以及搜索特定区域内的碎片。

可视化探索是系统中信息最丰富的模块，包含四个功能标签页，覆盖从宏观态势到微观追踪的全部需求。

标签页一：全球碎片态势

🌍 正射球面实时碎片动画

宏观态势 3D 动画

在一个可交互的正射球面投影上，实时展示碎片在地球表面的星下点运动轨迹。打开该标签页后，动画将自动播放，无需手动启动。

使用步骤

  1. 在左侧导航栏点击 「可视化探索」，默认进入第一个标签页
  2. 页面加载后，3D 地球将自动旋转并展示碎片的实时位置点
  3. 使用鼠标拖动旋转地球视角，滚轮缩放
  4. 利用侧边栏的筛选控件调整显示范围

筛选参数

参数说明默认值
高度范围仅显示近地点/远地点在指定范围内的目标0–2,000 km
目标类型选择显示碎片、载荷、箭体或全部全部
最大显示数量限制同时显示的目标数量以保持流畅度2,000

图例说明

球面上每个点代表一个在轨目标的当前星下点位置。不同颜色通常对应不同高度层或目标类型。碎片密集区域（如 LEO 带）会呈现出明显的带状分布。

标签页二：高度分层下钻

🏗️ 3D 地球 + 轨道层分层视图

分层分析 3D 交互

将轨道空间划分为多个高度层（VLEO / LEO / MEO / GEO 等），点击任意高度层即可查看该层内所有目标在 3D 球体上的分布。球体带有经纬线辅助定位，同样支持自动动画。

使用步骤

  1. 切换到第二个标签页 「高度分层下钻」
  2. 页面显示轨道层列表和各层目标数量统计
  3. 点击感兴趣的轨道层（如 LEO 450–2000 km）进行下钻
  4. 3D 球体自动渲染该层内的目标分布，支持拖拽旋转和缩放
  5. 动画自动播放，展示碎片的轨道运动

高度层划分

层级高度范围特征
VLEO0–450 km目标稀疏，大气阻力显著，衰减快
LEO450–2,000 km碎片密度最高，Starlink 集中区
MEO2,000–33,786 km导航卫星（GPS/北斗/Galileo）
GEO 带~35,786 km ± 2,000同步轨道通信/气象卫星
HEO37,786–120,000 km高椭圆轨道（闪电轨道等）

标签页三：目标轨道预报

🛰️ 单目标轨道传播与 3D 可视化

目标追踪 SGP4 传播 3D 轨迹

输入任意目标的 NORAD ID，系统将使用 SGP4/开普勒两体传播算法预测其未来轨道，并在 3D 地球上绘制完整的轨迹曲线。支持椭圆轨道和圆轨道的真实可视化。

使用步骤

  1. 切换到第三个标签页 「轨道预报」
  2. 在输入框中填写目标的 NORAD ID（如 25544 代表国际空间站）
  3. 设置传播时长（默认为一个轨道周期）
  4. 点击 「传播」 按钮
  5. 等待计算完成后，3D 地球上将显示完整的预报轨迹

输出内容

  • 3D 轨迹曲线：目标在 ECI 坐标系中的完整轨迹，投射到 3D 球体上
  • 星下点轨迹：目标在地球表面的投影路径
  • 轨道参数：周期、倾角、近地点/远地点高度等
  • OEM 导出：可导出标准 OEM 格式文件，用于外部工具对接
💡 提示：如果不知道目标的 NORAD ID，可以先到 目标目录 中按名称搜索，找到编号后再返回此处。

标签页四：区域碎片搜索

📍 地理位置附近碎片搜索

区域分析 空间查询 目标列表

指定一个地理坐标（如发射场位置），搜索该位置上空一定范围内的所有在轨空间目标。适用于发射前的空域安全评估或特定区域的态势监控。

使用步骤

  1. 切换到第四个标签页 「区域碎片搜索」
  2. 输入搜索中心的纬度和经度（如文昌发射场：19.61°N, 110.95°E）
  3. 设置搜索半径（km）和高度范围
  4. 选择目标类型筛选条件
  5. 点击 「搜索」，查看结果列表

搜索参数

参数说明默认值
纬度 (°)搜索中心的纬度19.61（文昌）
经度 (°)搜索中心的经度110.95（文昌）
搜索半径 (km)从中心点向外的搜索范围500
高度范围 (km)仅搜索指定高度区间内的目标0–2,000
目标类型ALL / DEBRIS / PAYLOAD / ROCKET BODYALL
时间窗口搜索时间范围当前 + 6 小时

搜索结果

结果以表格形式展示，每行包含：NORAD ID、名称、类型、国家代码、近地点、远地点、倾角、偏心率、与搜索中心的近似距离(km)。

💡 提示：区域搜索使用两种策略——当数据库中存在轨迹段数据时使用 PostGIS 空间索引快速查询，否则退化为 SGP4 实时传播逐目标计算。前者速度快但需要预先摄入轨迹数据，后者计算量较大但不依赖轨迹段。同样的功能也可通过 区域碎片查询 API 以编程方式调用。
