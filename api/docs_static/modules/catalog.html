目标目录

场景 查找特定的空间目标，或按条件批量筛选感兴趣的目标群体。

目标目录是系统的核心数据浏览界面，分为两个标签页：

  • 地球在轨目标（多源融合）：融合 Space-Track、UCS、ESA DISCOS 三大数据源，按 NORAD ID 去重后统一展示，支持多维度筛选和关键词搜索，最多一次显示 2,000 条记录。
  • 小行星 / NEO (Asterank)：独立的小行星 / 近地天体（NEO）专题库，字段包含开普勒轨道根数（a / e / i / MOID）、光谱类型、直径、Δv 与经济估值（price / profit），可按轨道族、光谱类型、Δv 上限、直径下限、最小利润筛选。

使用步骤

  1. 在左侧导航栏点击 「目标目录」
  2. 在页面顶部的 「筛选条件」 展开面板中设置搜索条件
  3. 条件设置后，结果表格将自动刷新
  4. 在结果表格中浏览数据，可点击列标题进行排序

搜索

系统支持多种搜索方式，可单独使用也可组合使用：

搜索方式说明示例
名称关键词模糊匹配目标名称（不区分大小写）输入 STARLINK 搜索所有 Starlink 卫星
国家代码精确匹配国家/地区代码PRC、US、CIS
近地点范围按近地点高度范围筛选200–600 km 查看 LEO 目标
💡 国家代码别名支持：输入 CN、China 或 中国 均可自动转换为 PRC；USA 或 美国 自动转换为 US；USSR、Russia 或 俄罗斯 自动转换为 CIS。

筛选条件

地球在轨目标（多源融合）

筛选项选项说明
目标类型全部 / DEBRIS / PAYLOAD / ROCKET BODY / UNKNOWN按空间目标分类筛选
国家代码自由输入如 US, PRC, CIS, JPN, IND
名称关键词自由输入模糊搜索，如 ISS, COSMOS
近地点下限数值 (km)默认 0
近地点上限数值 (km)默认 50,000
RCS 尺寸全部 / SMALL / MEDIUM / LARGE雷达截面积分类
主数据源全部 / Space-Track / UCS / ESA-DISCOS按主要数据来源筛选

小行星 / NEO（Asterank）

Asterank 标签页读取独立的 external_asterank 表，字段与地球在轨目标不同，提供以下筛选条件：

筛选项选项说明
名称关键词自由输入匹配 full_name 或 prov_des（临时编号），如 Bennu、2011 CP4
轨道族 class自由输入小行星轨道分类，如 APO（Apollo）、AMO（Amor）、ATE（Aten）、IEO（Atira）、MBA（主带）
光谱类型 spec自由输入按光谱分类筛选，如 S（硅酸盐）、C（碳质）、M（金属）、X、V
Δv 上限 (km/s)数值仅保留 Δv ≤ 阈值的天体（越小越易到达）；0 表示不限制
最小直径 (km)数值仅保留直径 ≥ 阈值的天体
最小估值 profit ($)数值（支持科学计数）仅保留 Asterank 经济估值 profit 高于阈值的天体

小行星 / NEO（Asterank）

列名数据来源含义
名称Asterank小行星正式名称（full_name）
临时编号Asterank未命名天体的临时编号（prov_des），如 2011 CP4
轨道族Asterank小行星轨道分类（class）：APO / AMO / ATE / IEO / MBA …
光谱Asterank光谱类型（spec）：S / C / M / X / V / D 等
半长轴 a (AU)Asterank (JPL SBDB)绕日椭圆轨道的半长轴（天文单位）
偏心率 eAsterank轨道偏心率，0 为正圆，接近 1 为细长椭圆
倾角 i (°)Asterank轨道面与黄道面的夹角
周期 (d)Asterank绕太阳公转周期（天）
直径 (km)Asterank估算的小行星直径
反照率Asterank表面反射率（albedo），反推尺寸的关键参数
MOID (AU)Asterank与地球轨道的最小交会距离，PHA（潜在威胁天体）判定依据
Δv (km/s)Asterank从地球出发到达该天体所需的速度增量，衡量「可达性」
估值 price ($)Asterank基于光谱和直径估算的资源总价值
利润 profit ($)Asterank扣除到达与返回成本后的净利润估值（综合考虑 Δv）
💡 提示：结果表格最多显示 2,000 条记录。如果需要处理更大量级的数据，建议使用筛选条件缩小范围，或通过 API 接口 编程获取数据。
