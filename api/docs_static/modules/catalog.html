目标目录

场景 查找特定的空间目标，或按条件批量筛选感兴趣的目标群体。

目标目录是系统的核心数据浏览界面，融合了 Space-Track、UCS 和 ESA DISCOS 三大数据源的信息，按 NORAD ID 去重后统一展示。支持多维度筛选和关键词搜索，最多可一次显示 2,000 条记录。

使用步骤

  1. 在左侧导航栏点击 「目标目录」
  2. 在页面顶部的 「筛选条件」 展开面板中设置搜索条件
  3. 条件设置后，结果表格将自动刷新
  4. 在结果表格中浏览数据，可点击列标题进行排序

搜索

系统支持多种搜索方式，可单独使用也可组合使用：

搜索方式说明示例
名称关键词模糊匹配目标名称（不区分大小写）输入 STARLINK 搜索所有 Starlink 卫星
国家代码精确匹配国家/地区代码PRC、US、CIS
近地点范围按近地点高度范围筛选200–600 km 查看 LEO 目标
💡 国家代码别名支持：输入 CN、China 或 中国 均可自动转换为 PRC；USA 或 美国 自动转换为 US；USSR、Russia 或 俄罗斯 自动转换为 CIS。
