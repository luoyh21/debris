碰撞风险评估

场景 对于一个已确定的发射时刻，评估火箭飞行各阶段各有多高的碰撞风险——回答「选定时刻的碰撞风险有多高，哪个阶段最危险」。

⚡ 前置条件：碰撞风险评估需要使用轨迹仿真的结果。请先在 轨迹仿真 页面运行仿真并生成轨迹数据。

工作原理

固定发射时刻后，系统将完整飞行轨迹划分为上升段、停泊轨道段等阶段，逐阶段与在轨碎片进行以下计算流程：

  1. 空间预筛选：使用 PostGIS 空间索引进行粗筛，选出轨迹附近的候选碎片（粗筛距离可配置）
  2. SGP4 传播：将候选碎片的 TLE 传播到飞行时刻，获取精确位置
  3. TCA 求解：通过三次样条插值求解火箭与每个碎片的最近接时刻 (Time of Closest Approach)
  4. Foster Pc 计算：在交汇平面上进行 2D 数值积分，计算碰撞概率
  5. 风险定级：根据 Pc 值划分风险等级，输出评估报告

使用步骤

  1. 确保已在 轨迹仿真 中运行仿真
  2. 在左侧导航栏点击 「碰撞风险」
  3. 在「评估参数」面板中调整联合硬体半径、筛选距离等参数
  4. 点击 「开始碰撞风险评估」
  5. 等待评估完成，查看各阶段风险摘要和详细合取事件表

评估参数

参数说明默认值
联合硬体半径 HBR (m)火箭 + 碎片外接球半径之和。Pc 计算的关键输入。20 m
载人任务勾选后 Pc 门限从 10⁻⁵ 提高到 10⁻⁶（更严格标准）未勾选
精细筛选距离 (km)在此距离内执行完整的 Foster Pc 计算50 km
粗筛距离 (km)在此距离内纳入 TCA 求解候选池200 km
注入演示威胁注入三条合成合取事件用于演示。碎片数据库较小时推荐开启开启

风险等级含义

等级标记Pc 范围含义建议操作
GREEN🟢Pc < 10⁻⁷极低风险，无需关注正常发射
YELLOW🟡10⁻⁷ ≤ Pc < 10⁻⁵低至中等风险关注但通常可接受
AMBER🟠10⁻⁵ ≤ Pc < 10⁻⁴中高风险，超过非载人门限建议调整发射时刻或方位角
RED🔴Pc ≥ 10⁻⁴高风险，必须规避必须更换发射窗口

📊 各阶段风险摘要

数据表 + 指标卡片

为每个飞行阶段（上升段、停泊轨道段等）显示：

  • 风险等级标记（GREEN/YELLOW/AMBER/RED）
  • 最大碰撞概率 (Max Pc)
  • 候选碎片数量
  • 检测到的合取事件数
  • 飞行时长和高度范围

💡 规避建议

文字

根据评估结果，系统自动生成规避建议。如果整体风险为 GREEN，建议正常发射；如果出现 AMBER 或 RED，会建议调整发射时刻（推荐使用 LCOLA 飞越筛选 找安全窗口）或调整发射方位角。

💡 提示：如果评估结果显示高风险，建议前往 LCOLA 飞越筛选 扫描整个发射窗口，找到所有安全发射时刻。两个模块配合使用效果最佳。
